Huzzah Workshop

For this workshop, please collect the following:

  • Huzzah board
  • Protoboard

First we need to configure Arduino IDE to work with the Huzzah. First, download and install the CP2104 USB Driver. Next, add this to Perferences > Additional Boards Manager URLs: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json. Finally, under Tools > Boards Manager, add the "esp32" boards package. Select Adafruit ESP32 Feather as your board.
