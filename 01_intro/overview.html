Course Overview

Scope and expectations

  • Many skill areas - broad and quick.
  • We intend for this course to be accessible to beginners as well as worthwhile for more advanced makers.
  • Each week is intended to let you get hands on and start an exploration.
  • After each week, you will know how to do something safely and correctly, and will have documented it. You will also identify resources for going further. This will give you what you need to return and learn more deeply later.
  • The final project is intended to show you how to integrate multiple capabilities. These topics add up in surprising ways. We will encourage you to build simple modular prototypes on the way to a final project. The result is not likely to be a finished product.

Grading

Grade is based on your documentation of learning, not on how much or what you learned.

  • Weekly documentation: 60%
  • Final project and its documentation: 20%
  • Participation (important due to safety concerns): 20%

Weekly pattern

Tuesdays in class:

  • Website review and discussion of work from prior week (breakout rooms). This should be the most intellectually nutritive part of the class, and is up to you to generate.
  • Preparation for new topic.
  • Assigned readings (Making Things Move, sparkfun.com, etc.)

Thursdays in class:

  • Lecture and demonstrations for new topic.
  • Hands-on training on new topic.

Thursdays - Tuesday:

  • Scheduled lab times. Multiple options, check in at least once.
  • Supplemental workshops.
  • Slack (channels, messaging).

Assignments

  • Common tasks.
  • Individual explorations.
  • Groupwork encouraged when possible.
  • Complete the assignment for a passing grade. Go above and beyond in terms of assignment scope and/or documentation for an excellent grade.

Weekly Overview

Week 1: Lab safety, documentation, software.

  • Introduce members of this class, Nathan, Victoria, Rob, and others in the lab.
  • Personal websites. Version control, GitHub, GH Pages.
  • Software choices:
    • Autodesk Fusion 360
    • Arduino
    • Inkscape, Figma, Illustrator

Week 2: 2D Design and Cutting. Electronics.

  • Fusion 360. Rhino, Freecad, others.
  • Laser cutter. File prep. Pre/post-processing.
  • Vinyl cutter.
  • “Assignment: Make a press-fit kit”: Kassia's triangles, Chris's pencil holder, Kian's Moose.
  • Circuits: Breadboards, current, voltage, resistance, Ohm’s law.

Week 3: Electronics. Hand tools and fabrication. Making things move. Mechanical design.

  • Electronics. Multimeters, TinkerCAD.
  • Nuts and bolts, bearings and belts, drilling, sawing, tapping, glueing.
  • Dustyn Miller: Making things move.
  • Assignment: Make a kinetic sculpture.

Week 4: Microcontroller Programming (Arduino IDE)

  • Microcontrollers. What are they? Which ones?
  • M0 boards Adafruit Metro M0 Express. Easy introduction, large set of possibilities.

  • Programming microcontrollers using Arduino environment (IDE).
  • Code for simple input and output. Buttons and LEDs.
  • Flow control: loops, conditionals.
  • Functions.
  • Serial communication basics.
  • Options for lower-level programming in C
  • Higher level programming: CircuitPython.

Week 5: 3D design. 3D scanning and printing.

  • Fusion 360 for 3D CAD.
  • Model creation via VR.
  • Several methods for scanning, photogrammetry.
  • Photogrammetry with Meshroom.

Two options for 3D printing:

  • Prusa MK3S
  • Form2

Week 6: Sensors. Electronic input devices.

  • Using Analog to Digital converter on microcontroller.
  • Capacitive methods. Touch and proximity.
  • Making sensors.

Week 7: Electronic output devices

  • Digital to analog conversion (DAC)
  • TFT Screens
  • Buzzers
  • RGB LED strips
  • Motors and other actuators

"Spring Break"

  • Output devices
  • Serial communication
  • ESP32 Huzzah
  • Ship 2nd packages

Week 8: Radio communication

  • Communication between microcontrollers
  • Radio based methods (ESP32 boards)
  • Point-to-point radio

Week 9: Networking (IoT)

  • ESP32-CAM
  • Bluetooth
  • WiFi
  • Firebase (serverless database)

Week 10: Machine building. End effectors.

  • Stepper motors and drivers. Simple command protocols.
  • Designs for machine axes
  • Machines that make

Week 11: Interfacing projects with computers. Computer programming.

  • Javascript, Processing, Python.
  • Sending data to a computer from your project.
  • Graphical display.
  • Controlling a project from a computer.

Week 12: Wildcard week

Molding and casting. Composites and sewing.

  • Wax -> Silicone -> plaster/metal.
  • Composites with epoxy and fiber.
  • Computer-controlled sewing.
  • Soft robotics.

Week 13: Project integration

  • Batteries, power supplies. Making a project robust.
  • Soldering a project using protoboards.
  • Making your own board to attach to microcontroller boards
  • Making your own microcontroller boards

Week 14: Project development

  • Integrating
  • Spiral development and project management

Week 15: Exam Week. Final Project Presentations.
