Assignment:

1. Use an output device that you haven't used before.
2. Write a microcontroller program that integrates at least one input device and one output device. Avoid the delay() function by using either timers or interrupts.
3. (Optional) Prepare a 3D model for molding and casting. You will each receive ~8oz of food-grade silicone. Consider designing a 3D-printable positive, which could be used to make a negative in silicone. You could then cast any number of materials into the negative. Please send any remaining 3D printing requests by Mar 21.

Examples: Kian (pan/tilt kit), Kassia (RGB LED / buzzer), Teddy (LED Strip / pot).
